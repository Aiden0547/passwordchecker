분석 설명

      

        

          이 분석기는 입력하신 비밀번호의 보안 강도를 무차별 대입 공격(Brute Force Attack)에 필요한 예상 시간을 기준으로 평가합니다.
        

        
        
주요 분석 요소

        
    
          
  • 비밀번호 길이: 자릿수가 늘어날수록 조합의 수는 기하급수적으로 증가합니다.
    • 
          
  • 문자 다양성: 대소문자, 숫자, 특수문자가 섞일수록 보안 강도가 높아집니다.
    • 
        


        
수학적 원리

        

          비밀번호 조합은 중복 순열의 원리를 따릅니다.
          사용 가능한 문자 종류를 $N$, 길이를 $L$이라고 할 때, 총 경우의 수는 다음과 같습니다.
        

        

          $$Total\ Combinations = N^L$$
